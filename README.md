# Local development setup

To do most things, you'll want both the frontend and backend running locally.

### First-time setup:

1. Clone this repo and [mathsoc-rentals-worker](https://github.com/MathSoc/mathsoc-rentals-worker).
2. Run `pnpm install` or `npm install` on each to install packages 

### Running the dev environment

1. Either `./run-worker.sh`, or run `npx wrangler dev` within your mathsoc-rentals-worker repository.
2. Run `npm run dev` locally
3. Open the URL `npm run dev` outputs
